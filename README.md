# 🏥 Medi-Bot: AI-Powered Drug Interaction Checker
**Medi-Bot** is a high-performance Python tool designed to enhance pharmaceutical safety. Leveraging the power of Google Gemini AI, it provides instant, data-driven insights into potential drug-drug interactions, helping users and researchers understand safety profiles in seconds.
## 👨‍💻 Developed by Hammad Virk
A Senior Health-Tech Developer focused on bridging the gap between Artificial Intelligence and modern Medicine.
## 🚀 Key Features
 * **AI-Driven Safety Analysis**: Uses Gemini 2.5 Flash for deep medical logic.
 * **Instant Interaction Checks**: Identifies severity levels (Minor, Moderate, Major) between two drugs.
 * **Side Effect Profiling**: Lists potential risks and physiological warnings.
 * **Clean CLI Interface**: Simple, lightweight, and efficient for developers and researchers.
 * **Reliability**: Built-in exponential backoff for API stability.
## ⚠️ Medical Disclaimer
**IMPORTANT: This software is for EDUCATIONAL and INFORMATIONAL purposes only.**
It does not provide medical advice. The information generated by Medi-Bot should not be used to diagnose or treat any health problem or disease. **Always consult with a licensed healthcare professional before making any decisions related to medication.**
## 🛠️ Installation
 1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/hammadvirk-dev/Medi-Bot.git](https://github.com/hammadvirk-dev/Medi-Bot.git)
   cd Medi-Bot
   
   ```
 2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   
   ```
 3. **Run the Application:**
   ```bash
   python medibot.py
   
   ```
